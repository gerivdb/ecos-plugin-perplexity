# Guide Complet des Modes Focus de Perplexity AI

## Introduction

Les **modes Focus** de Perplexity AI constituent l'une des fonctionnalités les plus puissantes pour optimiser vos recherches. Chaque mode cible des sources spécifiques et adapte le comportement de l'IA selon le type d'information recherchée. Au lieu de toujours chercher sur l'ensemble d'Internet, vous dirigez Perplexity vers les sources les plus pertinentes pour votre besoin.

---

## 1. Web Focus (Mode par défaut)

### Description
Le mode **Web Focus** effectue une recherche complète sur l'ensemble d'Internet, incluant tous types de contenus web.

### Sources utilisées
- Sites web généraux
- Blogs
- Articles de presse
- Contenus en ligne variés
- Bases de données publiques

### Quand l'utiliser
- Recherches générales et questions factuelles
- Vérification d'informations récentes
- Actualités et tendances
- Première exploration d'un sujet

### Comment l'employer
1. Cliquez sur le bouton **"Focus"** dans l'interface de recherche
2. Sélectionnez **"Web"** (activé par défaut)
3. Tapez votre question en langage naturel

### Exemples de requêtes optimales
```
"Comparez le régime méditerranéen et le régime DASH en termes simples"
"Dernières mises à jour sur les voitures électriques en 2025"
"Qu'est-ce que l'intelligence artificielle générative ?"
```

### Éviter quand
- Vous cherchez des sources académiques rigoureuses
- Vous voulez des citations scientifiques
- Vous avez besoin de calculs précis

---

## 2. Academic Focus

### Description
Mode spécialisé qui se concentre exclusivement sur les **sources académiques et scientifiques**, évitant le contenu généraliste du web.

### Sources utilisées
- Articles peer-reviewed
- Journaux scientifiques
- Bases de données académiques (PubMed, Semantic Scholar)
- Publications de recherche
- Thèses et mémoires universitaires

### Quand l'utiliser
- Rédaction d'essais et rapports
- Recherche universitaire
- Vérification de faits scientifiques
- Citations académiques nécessaires

### Comment l'employer
1. Cliquez sur **"Focus"**
2. Sélectionnez **"Academic"**
3. Formulez des questions précises et techniques

### Exemples de requêtes optimales
```
"Résumez les études récentes sur l'adoption des énergies renouvelables en Asie"
"Expliquez comment les microplastiques affectent la santé humaine, avec sources"
"Méta-analyse des effets du jeûne intermittent sur le métabolisme"
```

### Éviter quand
- Vous voulez juste une définition simple
- Vous cherchez des opinions publiques
- Vous avez besoin de contenu récent non-académique

---

## 3. Math Focus (Alimenté par Wolfram Alpha)

### Description
Mode spécialisé pour les **calculs mathématiques** et l'analyse de données, intégrant directement Wolfram Alpha.

### Sources utilisées
- Wolfram Alpha
- Bases de données computationnelles
- Sources mathématiques structurées
- Calculateurs scientifiques

### Quand l'utiliser
- Résolution d'équations complexes
- Conversions d'unités
- Calculs statistiques
- Analyses de données numériques
- Problèmes de mathématiques

### Comment l'employer
1. Sélectionnez **"Math"** dans Focus
2. Posez des questions numériques précises
3. Demandez des étapes de résolution si nécessaire

### Exemples de requêtes optimales
```
"Résolvez 3x² + 5x - 7 = 0 étape par étape"
"Listez les nombres premiers jusqu'à 500"
"Convertissez 200 kilomètres par heure en miles par heure"
"Calculez l'intégrale de x²sin(x) de 0 à π"
```

### Éviter quand
- Questions ouvertes ou créatives
- Recherche de tendances non-numériques
- Analyse qualitative

---

## 4. Writing Focus

### Description
Mode optimisé pour la **création de contenu** et les tâches rédactionnelles, incluant le code et le brainstorming.

### Sources utilisées
- Ressources de création de contenu
- Guides rédactionnels
- Exemples de code
- Templates et modèles

### Quand l'utiliser
- Génération de contenu créatif
- Rédaction d'articles ou blogs
- Développement de code
- Brainstorming d'idées
- Traduction de textes
- Correction grammaticale

### Comment l'employer
1. Activez **"Writing"** Focus
2. Spécifiez le type de contenu souhaité
3. Donnez le contexte et le style désiré

### Exemples de requêtes optimales
```
"Rédigez un email professionnel pour reporter une réunion"
"Créez un script Python pour analyser des données CSV"
"Traduisez ce texte de français vers anglais en gardant le ton formel"
"Générez 10 titres accrocheurs pour un article sur l'IA"
```

### Note importante
Pour les **traductions**, ce mode est **essentiel**. En mode "All", Perplexity cherche sur Internet au lieu de traduire directement.

---

## 5. Video Focus

### Description
Mode spécialisé dans la **recherche et analyse de contenu vidéo**, principalement YouTube, avec timestamps précis.

### Sources utilisées
- YouTube (prioritaire)
- Autres plateformes vidéo
- Transcriptions automatiques
- Métadonnées vidéo

### Quand l'utiliser
- Résumés de vidéos longues
- Recherche de tutoriels spécifiques
- Analyse de conférences
- Recherche de contenu éducatif visuel
- Éviter de regarder des heures de contenu

### Comment l'employer
1. Sélectionnez **"Video"** Focus
2. Mentionnez le type de contenu vidéo recherché
3. Demandez des timestamps si nécessaire

### Exemples de requêtes optimales
```
"Résumez les points clés de la conférence TED sur l'IA de 2024"
"Trouvez des tutoriels Python pour débutants avec timestamps"
"Que disent les experts sur l'iPhone 16 dans leurs reviews vidéo ?"
"Analysez les principales critiques du film Dune 2 sur YouTube"
```

### Limitation
Fonctionne uniquement quand les **transcriptions sont disponibles**.

---

## 6. Social Focus

### Description
Mode ciblant les **discussions communautaires** et opinions publiques sur forums et réseaux sociaux.

### Sources utilisées
- Reddit (prioritaire)
- Twitter/X
- Forums spécialisés
- Discussions communautaires
- Avis utilisateurs

### Quand l'utiliser
- Comprendre l'opinion publique
- Rechercher des expériences utilisateurs réelles
- Suivre les tendances émergentes
- Évaluer la réception d'un produit
- Détecter les discussions actuelles

### Comment l'employer
1. Activez **"Social"** Focus
2. Formulez des questions sur l'opinion ou l'expérience
3. Cherchez des retours authentiques

### Exemples de requêtes optimales
```
"Les lunettes Meta Rayban smart glasses valent-elles la peine selon les utilisateurs ?"
"Que pensent les gens de la nouvelle politique climatique sur Reddit ?"
"Expériences utilisateurs avec Tesla Model 3 en 2024"
"Réactions du public à la sortie de GPT-5"
```

### Éviter quand
- Vous cherchez des informations médicales fiables
- Vous voulez des faits vérifiés scientifiquement
- Questions nécessitant une expertise technique

---

## 7. Finance Focus (Pro uniquement)

### Description
Mode spécialisé dans les **données financières** et documents réglementaires officiels.

### Sources utilisées
- Documents SEC (Securities and Exchange Commission)
- Rapports financiers officiels
- Bases de données boursières
- Documents réglementaires

### Quand l'utiliser
- Recherche d'informations sur les entreprises cotées
- Analyse de performances financières
- Accès aux documents officiels
- Due diligence d'investissement

### Comment l'employer
1. Sélectionnez **"Finance"** (nécessite Pro)
2. Recherchez par ticker symbol ou nom d'entreprise
3. Spécifiez le type d'information financière

### Exemples de requêtes optimales
```
"Analysez les revenus de Tesla (TSLA) sur les 3 derniers trimestres"
"Comparez les bilans d'Apple et Microsoft en 2024"
"Trouvez les derniers documents SEC d'Amazon"
```

---

## 8. Pro Search (Mode Avancé)

### Description
Mode de **recherche approfondie** utilisant les modèles IA les plus avancés pour des analyses complexes multi-étapes.

### Caractéristiques
- Temps de réponse : 2-4 minutes
- Recherches multiples automatiques
- Vérification croisée des sources
- Synthèse complète et détaillée

### Quand l'utiliser
- Questions complexes nécessitant une analyse approfondie
- Recherches multi-dimensionnelles
- Rapports détaillés
- Comparaisons complexes

### Comment l'employer
1. Activez le slider **"Pro"** en bas à droite
2. Posez des questions complexes et ouvertes
3. Attendez l'analyse approfondie

### Exemples de requêtes optimales
```
"Analysez en détail les avantages et inconvénients de l'énergie solaire vs nucléaire"
"Déboguez ce code Python et expliquez l'erreur étape par étape"
"Rapport complet sur l'impact économique de l'IA générative en 2025"
```

## Stratégies de Combinaison des Modes

### Recherche Académique Complète
1. **Academic Focus** → Trouvez les études de base
2. **Web Focus** → Vérifiez le contexte actuel
3. **Writing Focus** → Rédigez la synthèse

### Évaluation de Produit
1. **Web Focus** → Informations techniques officielles
2. **Social Focus** → Expériences utilisateurs réelles
3. **Video Focus** → Reviews détaillées

### Recherche Technique
1. **Math Focus** → Calculs et données précises
2. **Academic Focus** → Validation scientifique
3. **Writing Focus** → Documentation et code

---

## Conseils d'Optimisation

### Sélection du Mode
- **Commencez large** (Web) puis **affinez** (Academic/Social)
- **Adaptez selon le public cible** de votre recherche
- **Changez de mode** si les résultats ne correspondent pas

### Formulation des Questions
- **Soyez spécifique** sur le type d'information recherchée
- **Mentionnez le contexte** quand nécessaire
- **Demandez des formats précis** (listes, comparaisons, étapes)

### Suivi et Approfondissement
- **Utilisez les questions de suivi** suggérées
- **Combinez plusieurs modes** pour une vision complète
- **Sauvegardez dans Collections** pour réutilisation

---

## Limitations à Connaître

- **API Perplexity** : Pas de sélection Focus possible
- **Mobile** : Modes Focus limités selon la version
- **Transcriptions vidéo** : Nécessaires pour Video Focus
- **Abonnement Pro** : Requis pour Finance Focus et fonctionnalités avancées

---

Cette maîtrise des modes Focus transforme Perplexity d'un simple moteur de recherche en assistant de recherche intelligent, capable de s'adapter précisément à vos besoins spécifiques.
